# Apps

Collection of React apps from the [Full Stack Open](https://fullstackopen.com/en/part2) course (Part 2). Each subfolder is a standalone Vite + React app.

| App | Description |
|-----|-------------|
| [courseinfo](./courseinfo) | Displays course and exercise information |
| [phonebook](./phonebook) | Phonebook contact list |
| [notes](./notes) | Notes app |

Run any app from its folder: `npm install` then `npm run dev`.
